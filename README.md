## 介绍

基于bootstrap5.1.x设计的一个静态后台管理HTML模板，旨在快速让喜欢用bootstrap开发的后台程序员有个愉悦的起点。

[>>>演示地址](https://ajiho.gitee.io/bootstrap-admin)

## 特性

- 高度响应式、一站匹配手机、平板、pc
- 整体白色为基调、简约、大气
