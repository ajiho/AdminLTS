AdminLTS是一个基于[bootstrap(5.1.3+)](https://getbootstrap.com/)风格简约的纯静态通用后台模板项目,顾名思义(LTS),它是一个安全可靠的长期支持和维护的后台界面模板解决方案。无论您是用于项目开发、学习、毕设、还是教学视频演示、希望能给个star，这将会是我最大的动力!
如果对您有所帮助，您可以点仓库右上角 "Star" 收藏一下 ，获取第一时间更新，谢谢！
