# 依赖和插件


AdminLTS精选了以下插件。有关文档、更新或许可信息，请访问提供的链接,感谢以下项目为开源生态做出的贡献,您如果觉得哪些插件好用也可以通过pr的方式添加到该页面中

::: danger 建议 
对于推荐的插件请尽量选择提交活跃、不依赖jQuery的插件,详情参阅:[路线图](/guide/roadmap)
:::
