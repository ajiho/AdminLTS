bootstrap-admin也提供了一些css助手类,方便我们使用,因此在页面中看到如下类名不要觉得奇怪,因为bootstrap-admin的助手类都是bsa-开头

cursor ​

鼠标手势:分别是让鼠标指针变成手指、和让手指变成鼠标默认指针

  • .bsa-cursor-pointer
  • .bsa-cursor-default

ellipsis ​

文本省略助手,可以让文本保持指定的行数,多余的内容用省略号代替

.bsa-ellipsis-{1-5}

fontsize ​

快速设置字体大小

.bsa-fs-{25-60}

opacity ​

设置透明度

.bsa-opacity-{1-9}

radius ​

设置圆角

.bsa-radius-{10-30}

spacing ​

设置宽高

.bsa-h-{1-20}
.bsa-w-{1-20}
.bsa-wh-{1-20}
